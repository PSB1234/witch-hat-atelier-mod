# Witch Hat Atelier Mod

> ⚠️ **Prototype / Work in Progress** — This mod is an early, experimental prototype. Features are incomplete, unbalanced, and subject to change or removal without notice. Expect bugs, missing textures, and breaking changes between versions. Not recommended for use in a survival world you care about.

## About

**Witch Hat Atelier Mod** is a fan-made Minecraft mod that brings the magic system from the manga series *Witch Hat Atelier* (created by Kamome Shirahama) into Minecraft. The goal is to let players draw, cast, and experiment with a Minecraft-flavored take on the manga's sigil-and-spell-circle style of magic.

This is an unofficial fan project made for fun and learning. It is **not affiliated with, endorsed by, or sponsored by** Kamome Shirahama, Kodansha, or any other official rights holder of *Witch Hat Atelier*. All rights to the original names, characters, and concepts belong to their respective owners.

## Current Status

This project is in an early prototype stage. Right now that means:

- Core mechanics are still being designed and may not reflect the final vision of the mod.
- Content (items, blocks, spells, recipes) is minimal and mostly for testing purposes.
- APIs, item IDs, and data formats may change between commits without backward compatibility.
- No promises are made about stability, performance, or multiplayer support yet.

## Planned Features

The long-term goal is to bring in mechanics inspired by the manga's magic system, such as:
 
- Drawing spell circles / sigils to cast spells
- A "paper first" magic system — sigils and spells are drawn/practiced on paper before they can be cast in the world, mirroring how apprentices in the manga practice on paper before casting real magic
- Ability to draw anywhere in world to cast spells
- A "brush and ink" style casting system
- Witch hats and apprentice robes as wearable items
- Spellbooks and recipes for learning new magic
- A Witch Hat Atelier ecosystem — ateliers, apprentices, and other world elements from the manga's setting

*(Planned features are aspirational and may change as development progresses.)*

## Installation

Since this is a prototype, there is currently no stable release recommended for general use. If you'd like to try it anyway:

1. Clone or download this repository.
2. Check the mod loader and Minecraft version requirements (see the project files for current targets).
3. Build the mod or use a provided build if available.
4. Drop the resulting jar into your `mods` folder.

Expect things to break. Please back up your worlds before testing.
